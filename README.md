# CI4 Authentication Project

A complete authentication system built with CodeIgniter 4.

## Features

* User Registration
* Email OTP Verification
* User Login
* Password Hashing
* Session Management
* Multiple Image Upload
* Form Validation

## Technologies Used

* PHP 8.2
* CodeIgniter 4
* MySQL
* Bootstrap

## Installation

1. Clone the repository

```bash
git clone https://github.com/purnendumondal123/ci4-authentication-project.git
```

2. Install dependencies

```bash
composer install
```

3. Create `.env` file from `env`

4. Configure database credentials

5. Import database SQL file

6. Start development server

```bash
php spark serve
```
